Swagger

Swagger es una herramienta utilizada para diseñar, construir, documentar y utilizar servicios web RESTful. Permite a los desarrolladores y usuarios entender y trabajar con una API de servicio web de una manera más fácil y visual.

Desde swagger se pueden realizar las llamadas a la propia API y comprobar su correcto funcionamiento sin necesidad de otras aplicaciones como Postman.
